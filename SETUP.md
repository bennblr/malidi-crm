# Инструкция по запуску проекта

## Выполненные шаги

✅ Установлены все зависимости (`npm install`)
✅ Создан файл `.env` с настройками базы данных
✅ Синхронизирована схема базы данных (`prisma db push`)
✅ Заполнена база данных начальными данными (`prisma seed`)

## Запуск проекта

Проект уже запущен! Сервер разработки работает на http://localhost:3000

### Доступ к системе

1. Откройте браузер и перейдите на http://localhost:3000
2. Вы будете перенаправлены на страницу входа
3. Используйте следующие учетные данные:
   - **Email**: admin@example.com
   - **Пароль**: admin123

## Структура приложения

- **/login** - Страница входа
- **/board** - Канбан доска с заявками
- **/settings** - Настройки системы (приоритеты, колонки, общие настройки)
- **/users** - Управление пользователями (только для администраторов)

## Настройка Telegram бота

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Откройте файл `.env` и замените `your-telegram-bot-token-here` на ваш токен
4. Перезапустите сервер

## Важные замечания

- База данных уже настроена и заполнена
- Дефолтные приоритеты: normal (зеленый), warn (желтый), crit (красный)
- Дефолтные колонки: "Новые заявки", "В работе", "Выполнено"
- Системный пользователь для Telegram бота: system@telegram.bot

## Команды для управления

- `npm run dev` - Запуск сервера разработки
- `npm run build` - Сборка для продакшена
- `npm run prisma:studio` - Открыть Prisma Studio для просмотра БД
- `npm run prisma:seed` - Повторно заполнить БД начальными данными
